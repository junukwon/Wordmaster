# WordMaster 컴팩트 홈 DAY 그리드 설계

## 목적

iPad Mini 가로 화면에서 DAY 20개를 한 화면에 빠르게 훑고 선택할 수 있게 홈 화면을 압축한다. DAY가 100개까지 늘어나도 화면 밀도와 탐색 방식이 변하지 않아야 한다.

## 승인 시안

사용자가 승인한 최종 목업의 구조를 구현 기준으로 사용한다.

- 5열 × 4행의 컴팩트 DAY 카드
- 20 DAY 단위 구간 필터
- DAY 번호·주제 검색
- 숙달·학습 중·미학습 3색 범례와 작은 숫자 캡슐
- 하단 학습 시작·수시 테스트 고정 액션
- 상단 학습 범위 설정 진입 영역

## 홈 정보 구조

1. 얇은 브랜드 헤더에는 WordMaster, 설명, 오늘 복습 수, 발음 설정을 배치한다.
2. 묶음·범위·랜덤 학습 진입 영역은 DAY 그리드 위에 유지한다.
3. DAY 선택 제목 아래에 검색창, 상태 범례, 20개 단위 구간 버튼을 한 줄 도구 모음으로 배치한다.
4. 선택된 구간과 검색 조건에 맞는 DAY 카드만 표시한다.
5. 선택 요약과 학습 시작·수시 테스트 버튼은 화면 하단의 하나의 액션 영역에 표시한다.

## DAY 구간과 검색

- DAY를 20개 단위로 묶는다: 01–20, 21–40, 41–60, 61–80, 81–100.
- 실제 데이터에 존재하는 구간만 생성한다. 현재 DAY 01–20만 있으면 구간 버튼도 하나만 표시한다.
- 검색어가 비어 있으면 선택된 구간의 DAY만 표시한다.
- 검색어가 있으면 전체 DAY에서 DAY 번호 또는 주제를 검색한다. 검색 중에는 구간 제한을 적용하지 않는다.
- 검색 결과가 없으면 `검색 결과가 없어요.`를 표시한다.
- 그리드에서 보이지 않는 DAY의 기존 선택은 유지한다. 하단 선택 수와 신규 단어 수에도 계속 포함한다.

## 컴팩트 DAY 카드

- iPad 가로 화면에서 5열을 사용하고 카드 최소 높이는 약 108px로 줄인다.
- 카드에는 DAY 번호, 한 줄 주제, 선택 원, 3개 상태 숫자만 표시한다.
- 전체 단어 수와 `숙달 0 / 학습 중 0 / 미학습 25` 세로 문장은 제거한다.
- 상태 숫자는 왼쪽부터 숙달, 학습 중, 미학습 순서다.
- 상태 색은 기존 학습 평가와 동일하게 사용한다.
  - 숙달: `#2A9D8F`
  - 학습 중: `#E9C46A`
  - 미학습: `#E76F51`
- 카드 안에서는 상태명을 반복하지 않고 숫자만 보여준다. 도구 모음의 단일 범례가 의미를 설명한다.
- 각 캡슐은 약 34–38px 폭, 16–18px 높이의 작은 표시로 구현한다.
- 스크린리더용 카드 이름에는 각 상태명과 수치를 모두 포함한다.

## 반응형 동작

- iPad Mini 가로 크기에서는 5열 × 4행과 하단 액션이 가로 폭 안에 들어와야 한다.
- 더 좁은 세로 화면에서는 2열 또는 3열로 자연스럽게 줄어든다.
- 카드와 주요 버튼의 터치 영역은 최소 44px을 유지한다.
- 가로 스크롤은 발생하지 않아야 한다.

## 구성 요소

- `HomeDayToolbar`: 검색, 상태 범례, DAY 구간 선택을 담당한다.
- `DaySelectionGrid`: 전달받은 DAY만 렌더링하고 선택 토글을 담당한다.
- `HomePage`: 검색·구간 상태와 필터링, 전체 선택 요약을 담당한다.
- 검색과 구간 계산은 순수 함수로 분리해 단독 테스트한다.

## 기존 동작 보존

- 선택한 DAY들의 오름차순 정렬과 정확한 학습 세션 생성
- 진행 중 세션 교체 확인
- 묶음·범위·랜덤 학습 화면 진입
- 수시 테스트 진입
- 발음 설정
- localStorage 학습 기록과 sessionStorage 학습 범위 설정 복원
- PWA와 오프라인 동작

## 테스트

- 45개 DAY에서 01–20, 21–40, 41–45 구간을 계산한다.
- 선택된 구간의 DAY만 표시한다.
- DAY 번호와 주제 검색은 전체 DAY에서 동작한다.
- 카드에 세 상태 숫자와 접근 가능한 전체 상태 설명이 표시된다.
- 필터 밖의 선택도 유지되고 하단 선택 요약에 반영된다.
- iPad Mini 가로 E2E에서 페이지 가로 넘침이 없고 5열 카드가 표시된다.
- 기존 학습 시작, 세션 교체, 수시 테스트 E2E가 계속 통과한다.

## 제외 범위

- DAY 데이터를 서버에서 불러오기
- 가상 스크롤
- 사용자가 구간 크기를 변경하는 설정
- 카드 정렬 순서 변경
- 학습 진도 저장 형식 변경
